   ────────────────────────────

   - 탭 순서: [📊 포트폴리오] [🏦 통합 대시보드]
   - 통합 대시보드가 우측에 위치

   변경 후:
   - 탭 순서: [🏦 통합 대시보드] [📊 계좌1] [📊 계좌2] ...
   - 통합 대시보드가 좌측 끝(첫 번째)으로 이동

2. 포트폴리오 탭 → 계좌별 개별 탭으로 분리
   ──────────────────────────────────────────
   변경 전:
   - 단일 "📊 포트폴리오" 탭 하나만 존재
   - 탭 클릭 시 activePortfolioId 기준 포트폴리오 표시

   변경 후:
   - 각 계좌(포트폴리오)가 별도의 탭으로 표시
   - portfolios 배열을 map()하여 탭을 동적으로 렌더링
   - 계좌 추가 시 해당 계좌 탭이 자동으로 상단에 생성
   - 계좌 탭 클릭 시 switchToPortfolio(id) 호출하여 해당 계좌로 전환

   수정 내용 (src/App.tsx):
   - 뷰 전환 탭에 portfolios.map() 추가
   - 활성 계좌는 title state 기준으로 이름 표시 (실시간 반영)
   - 비활성 계좌는 portfolios 배열의 name 표시


3. 계좌 이름 변경 시 탭 이름 실시간 반영
   ──────────────────────────────────────────
   동작:
   - 포트폴리오 뷰에서 계좌 이름(헤더) 수정 시
   - 상단 탭 이름도 즉시 업데이트

   구현 방식:
   - 탭 렌더링: p.id === activePortfolioId ? title : p.name
   - title state와 portfolios 배열 name을 updatePortfolioName에서 동시 업데이트
     → 자동으로 동기화됨
