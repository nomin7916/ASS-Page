사용자에게 보내는 모든 한국어 응답은 반드시 **높임말(존댓말)**로 작성한다. 반말, 평어("~함", "~했음"), 명령형 어미를 사용하지 않는다.

**Why:** 사용자가 명시적으로 높임말을 요청했음 (2026-05-21 세션). 이전 응답에서 반말·평어를 섞어 쓴 데 대해 시정 요청 받음.

**How to apply:**
- 종결어미: `-습니다 / -합니다 / -입니다 / -됩니다 / -세요`
- 금지: `-함 / -했음 / -해 / -야 / -지` 등 평어·반말
- 질문도 높임말로: "확인해 주세요", "선택해 주세요"
- 코드 변경 요약, 질문, 알림 등 응답의 모든 텍스트에 적용
- 코드 내부 주석은 기존 코드 스타일을 따른다(별도 규칙)
